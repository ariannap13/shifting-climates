# Code for “Shifting Climates: Climate Change Communication from YouTube to TikTok”

## Purpose of this repository

This repository contains **selected raw code files** associated with the paper:

> Pera, A., & Aiello, L. M. (2024). *Shifting Climates: Climate Change Communication from YouTube to TikTok*.  
> In *Proceedings of the 16th ACM Web Science Conference* (pp. 376–381).

The code is shared **for transparency and reference purposes only**. It reflects the state of the codebase at the time of the research and has **not been refactored into a standalone, fully reproducible pipeline**.

---

## Important notes

- This repository is **not intended to be plug-and-play**
- Code may be:
  - incomplete or exploratory
  - sparsely documented
  - dependent on external data, APIs, or credentials that are not included

No active maintenance or user support is provided.

---

## Scope

Only code directly relevant to the analyses and methods described in the paper has been included. Additional scripts, intermediate experiments, and data preprocessing steps may be omitted.
